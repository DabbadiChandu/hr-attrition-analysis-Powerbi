# HR Attrition Analysis — Power BI Project

## About this project
This is my second data analytics project as a fresher.
I analysed HR employee attrition data across 5 departments
to identify why employees are leaving and recommend solutions.

## Tools used
- Microsoft Excel (data cleaning, pivot tables, Age Group formula)
- Power BI Desktop (interactive dashboard, DAX measures)

## Dataset
100 employee records — 5 departments — Jan to Mar 2024

## Key findings
- Overall attrition rate: 45% (critical level)
- Sales department: 100% attrition — main reason: Low Salary
- Marketing department: 67% attrition — main reason: Work-Life Balance
- 73% of all leavers are under 35 years old

## What I did
- Cleaned 6 data errors including typos, blanks and wrong values
- Created Age Group column using nested IF formula
- Built 2 pivot tables — by Department and by Age Group
- Built Power BI dashboard with 5 visuals and 1 DAX measure
- Wrote 3 data-driven recommendations for HR manager

## Recommendations
1. Revise Sales salary structure — increase by minimum 20-25%
2. Introduce flexible working policies for Marketing team
3. Launch young employee retention programme for under-35 staff
